SORTING

The sorting project was a good way to learn about some of the diffrent sorting methods that there are the ones that we used were bubble sort, selection sort, and merge sort. Each one that we used we had to keep track of the comparisons that were made within each of the diffrent methods and the print out the results of each method with a diffrent ammounts of array lengths.
